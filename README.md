# Trello Mini-Clone

## Quick Start
1. Ensure Docker Desktop is running.
2. In the root directory: `docker-compose up --build`
3. UI: http://localhost:4200
4. API Swagger: http://localhost:8080/swagger-ui/index.html

## Local Dev (without Docker)
1. Backend: `cd backend && ./mvnw spring-boot:run`
2. Frontend: `cd frontend && npm install && npm start`
